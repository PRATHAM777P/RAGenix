# Welcome to RAGenix 🔍🤖

**RAGenix** is an Advanced Retrieval-Augmented Generation (RAG) agent that lets you have intelligent conversations with your PDF documents.

## How to get started 🚀

1. **Upload a PDF** — click the upload button and select your document(s)
2. **Wait for processing** — RAGenix will index your content using hybrid search
3. **Ask questions** — get precise answers with source citations

## Features ✨

- 🔀 **Hybrid Search** — combines keyword and semantic search for better recall
- 🎯 **Cohere Reranking** — surfaces the most relevant passages
- 💬 **Conversation Memory** — ask follow-up questions naturally
- 📌 **Source Citations** — every answer links back to the original text

## Tips 💡

- Upload up to **3 PDFs** at once to compare documents
- Ask specific questions for the most accurate results
- Follow-up questions work — the agent remembers context
